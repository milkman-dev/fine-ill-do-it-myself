# Fine, I'll Do It Myself

> *Couldn't find the theme I wanted, so here we are*

Dark VS Code theme with a vivid pastel palette. High contrast where it matters, easy on the eyes everywhere else. Charcoal background, colorful syntax, no eye strain.

---

## The Palette

| Token | Color |
|---|---|
| Keywords (`def`, `func`, `class`, `let`) | Sky blue `#61AFEF` |
| Control flow (`return`, `if`, `for`, `while`) | Radiant pink `#D291FF` |
| Functions & methods | Pastel yellow `#E5D0A1` |
| Types, classes & interfaces | Teal green `#6EC9A8` |
| Variables & properties | Pale cyan `#9CDCFE` |
| Parameters | Pale cyan `#9CDCFE` *(italic)* |
| Numbers & booleans | Sage green `#98C379` |
| Strings | Peach orange `#D19A66` |
| Operators (`=`, `!=`, `<=`, `&&`) | Near-white `#D9E0EE` |
| Decorators, attributes & lifetimes | Purple `#C678DD` |
| Comments | Slate gray `#5C6370` *(italic)* |
| Editor background | Deep charcoal `#282C34` |
| Sidebar & panels | `#21252B` |
| Cursor & selection | Pink `#D291FF` |

---

## Install

**From the Marketplace** — search `Fine ill do it myself` in the Extensions panel (`Ctrl+Shift+X`).

**Or build it locally:**
```sh
vsce package
code --install-extension fine-ill-do-it-myself-0.1.0.vsix
```

**Activate:** `Ctrl+K Ctrl+T` → select **Fine ill do it myself**
